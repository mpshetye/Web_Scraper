## General info
Program to get data from a shopping site(Flipkart) and a video sharing website(YouTube) by webscraping
	
## Technologies
Project is created with:
* NodeJs v14.15.0
* npm v6.14.8
* Nodemom ^2.0.7v
*axios ^0.21.1v
*cheerio ^1.0.0-rc.9v
*convert-array-to-csv ^2.0.0v
*json2csv ^5.0.6v
*puppeteer ^9.1.1v

	
## Setup
To run this project, install it using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
